# Dominus-RPG
_A Role-Playing Game server for San Andreas Multiplayer (SA-MP)_

![Work in progress](https://img.shields.io/badge/Status-Work_in_progress-purple)


---

## 📖 About
Dominus-RPG is a private project to create a **Role-Playing Game server** for  
**Grand Theft Auto: San Andreas Multiplayer (SA-MP)**.  

The goal is to provide an immersive RPG experience similar to the real world,  
with jobs, properties, vehicles, economy, and player interactions.  

It is currently implemented on version **0.3.DL-2-MP**.
